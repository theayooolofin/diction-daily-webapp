# Diction Daily

A small browser web app to learn and pronounce 5 new words every day.

## Features

- Daily set of 5 words (changes each date)
- Generate a fresh set of 5 words any time
- History of all generated 5-word sets
- Word, part of speech, meaning, and example sentence
- Written pronunciation for each word
- Audio pronunciation using browser text-to-speech
- Speaking check using microphone speech recognition
- Daily progress saved in local storage

## Run

Open `index.html` in a modern browser (Chrome or Edge recommended for speech recognition).

## Notes

- If speech recognition is unavailable in your browser, use `Mark practiced` to continue.
- `Reset today` clears progress for the current day only.
